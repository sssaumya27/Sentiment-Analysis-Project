# Sentiment Analysis Project

This project predicts whether a movie review is **Positive or Negative** using a trained **LSTM deep learning model**.

## Live Demo
https://sentiment-analysis-project-knekt62fnmaqspvpugiqb7.streamlit.app/

## Features
- Sentiment prediction for movie reviews
- Built using LSTM model
- Streamlit web interface
- Real-time prediction

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- Scikit-learn
- NLTK
- NumPy
- Pandas
